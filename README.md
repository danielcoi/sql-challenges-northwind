# 🚀 Proyecto: Desafíos SQL - Caso Northwind

Este repositorio contiene las soluciones a un conjunto de requerimientos analíticos sobre la base de datos **Northwind**. 

### 🛠️ Lo que vas a encontrar en este proyecto:
Si revisás mi archivo `Query_1.sql`, vas a ver la aplicación práctica de conceptos clave de bases de datos relacionales:

1.  **Filtros Avanzados (`WHERE`):** Extracción de datos específicos para campañas de Marketing.
2.  **Lógica de Inventario (`AND`, `OR`):** Consultas críticas para el área de Operaciones y Stock.
3.  **Agrupaciones y Métricas (`GROUP BY`, `COUNT`):** Análisis de distribución geográfica para Recursos Humanos.
4.  **Conexión de Tablas (`INNER JOIN`):** Consolidación de registros de ventas para Finanzas.

*Nota: Podés ver el código completo y optimizado ejecutando el archivo `Query_1.sql` que se encuentra en la raíz de este repositorio.*
